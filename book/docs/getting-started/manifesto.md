# The ATON Manifesto

Engineering is not about documents.

Engineering is about knowledge.

Knowledge should not be trapped inside proprietary tools.

Knowledge should be open.

Knowledge should be versioned.

Knowledge should be connected.

Knowledge should be reviewable.

Knowledge should remain understandable by humans.

ATON is built on a simple belief:

Engineering deserves a common operating system.

An operating system that does not replace engineers.

An operating system that empowers engineers.

An operating system where every artifact has a history.

Every relationship has meaning.

Every decision is traceable.

Every capability builds upon the same foundation.

We believe in openness.

We believe in simplicity.

We believe that architecture matters more than features.

Because great engineering starts with great foundations.
