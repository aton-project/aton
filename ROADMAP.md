# ATON Roadmap

The roadmap describes the planned evolution of ATON.

---

# v0.1 Foundation

## Goal

Establish the architectural foundation of the project.

### Scope

- Vision
- Design Principles
- Architecture Decision Records
- Docusaurus Book
- Branding
- GitHub Organization
- Repository Structure

---

# v0.2 Kernel

## Goal

Define and implement the Kernel.

### Scope

- Engineering Domain Model
- EMF Metamodel
- Artifact
- Relation
- Version
- Review
- Baseline
- Kernel Services

---

# v0.3 Persistence

## Goal

Implement Git-based persistence.

### Scope

- Repository Layout
- Serialization
- Git Integration
- Version History

---

# v0.4 REST API

## Goal

Expose the Kernel through a REST API.

### Scope

- CRUD Operations
- OpenAPI Specification
- Authentication
- Queries

---

# v0.5 Web

## Goal

Provide a modern web application.

### Scope

- Artifact Browser
- Graph Explorer
- Search
- Review UI

---

# v0.6 AI

## Goal

Integrate AI-assisted engineering capabilities.

### Scope

- Semantic Search
- Knowledge Assistant
- AI-powered Queries
- Engineering Recommendations

---

# Future

Possible future capabilities include:

- Requirements Management

- Risk Management
- Project Management
