# Security Policy

Thank you for helping keep ATON secure.

## Reporting a Vulnerability

Please do not disclose security vulnerabilities publicly.

Instead, report them privately by email:

contact@aton-project.org

Include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

We will acknowledge your report as soon as possible.

## Supported Versions

At this early stage of the project, only the latest development version is supported.

## Responsible Disclosure

Please allow reasonable time for investigation and remediation before publicly disclosing vulnerabilities.
